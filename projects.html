Stealth Web-Based Canary Trap

#websecurity #detection #osint March 2025

This project implements a passive, stealth-based canary trap using a 1x1 transparent image embedded in a public-facing cert showcase. The trap silently triggers when a visitor loads the page, logging metadata (IP, UA, referer) to a Discord webhook for real-time alerting.

  • Tech Used: Cloudflare Workers, Discord Webhooks, HTML/CSS
  • Key Feature: Passive detection via image load without interaction
  • Purpose: Recon and monitoring of traffic to sensitive site elements
